## Tasca S11.01. Visualització de Dades amb Python i Power BI

En esta sección se trabaja con el dataset de la compañia: Galaxy & Throne,
dedicada al comercio de merchandasing de temática fantástica y de ciencia ficción.

Se ha realizado un notebook de Python sobre las ventas comprendidas entre los años 2015 a 2024,
importando los datos mediante una conexión con SQL y generando gráficos usando las librerías 
Matplotlib, Seaborn y Plotly; adicionalmente se han generado los mismos en Power BI usando 
scripts de Python adaptándolos a este entorno.
